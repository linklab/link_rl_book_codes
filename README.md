### 1. 환경구성
- conda create -n link_rl_book_codes python=3.7
- conda activate link_rl_book_codes
- git clone
- cd ~/git/link_rl_book_codes/

### 2. Package 설치
- pip install tensorflow==2.1.0
- pip install gym

### 3. requirements.txt 구성 
- pip freeze > requirements.txt
